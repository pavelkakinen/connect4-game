# Connect 4 Game

A fully functional Connect 4 game built with C# implementing clean architecture principles.

##  About The Project

This project demonstrates professional software development practices through a classic Connect 4 game implementation. The focus is on clean code architecture, modularity, and maintainability.
##  Features

-  Console and web-based gameplay
-  Clean architecture with separation of concerns
-  JSON/database-driven game state management
-  Full CRUD functionality for game sessions
-  Object-oriented design patterns
-  Modular and testable code structure

##  What I Learned

- Implementing clean architecture in a real project
- Designing modular, maintainable code structures
- Working with databases and JSON serialization
- Applying object-oriented design patterns
- Building both console and web interfaces for the same logic
